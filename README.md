Welcome to your new dbt project!

### Setup
1. Create python environment and pip install dbt-databricks
2. Run dbt deps (packages cuyrrently)
3. Get databricks token and set in environment var DATABRICKS_TOKEN

### Using the starter project

Try running the following commands:
- dbt run
- dbt test
